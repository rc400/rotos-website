# ROTO's TCG — Design System

You are building a landing page for a Pokémon TCG vending brand with a vintage, immersive visual identity. The site should feel like entering a world — mysterious, premium, with a sense of unknown lore behind the brand. Think physical products rooted in fantasy (Pokémon). NOT vibe-coded. No purple gradients, random emojis, or generic startup UI.

## Creative Direction

Inspired by:
- **BitcoinOS (bitcoinos.build)** — scroll animations, dynamic background, "world with unknown lore" feeling, dramatic section reveals, bold typography that commands attention
- **Structured (structured.app)** — clean layout with personality, strong social proof, smooth transitions, content that breathes

Key principles:
- The site should feel like a **world to explore**, not a brochure to read
- Scroll-triggered animations to reveal content dramatically
- Dynamic/subtle background textures or effects (not static flat pages)
- Bold, confident typography — the brand has presence
- Product imagery does the heavy lifting
- The cream + black + burnt orange palette feels vintage and premium

## Color Palette
- Primary: #040707 — near-black, used for text, outlines, key UI elements
- Secondary: #FFF8E5 — warm cream, main canvas and card backgrounds
- Accent: #D56C3F — burnt orange (from ROTO's banner), used for CTAs, highlights, hover states
- Accent Dark: #CB673D — deeper orange for hover/active states
- Teal (supporting): #376B76 — from brand illustrations, use sparingly for contrast
- Background: #FFF8E5 — warm cream canvas
- Surface: #FFFFFF — cards, modals, elevated elements on cream background
- Text Primary: #040707 — headings, body copy
- Text Secondary: #5A5652 — captions, helper text, muted content
- Border: #CEC7BB — dividers, input outlines (warm gray)
- Success: #2D6A4F
- Warning: #D56C3F — matches accent
- Error: #C1292E

## Typography
- Font Family: Space Grotesk (headings — geometric with retro character) + DM Sans (body — clean and readable)
- Headings: bold, tracking tight, can go large and dramatic
- Body: regular weight, generous line height
- Size Scale: 12 / 14 / 16 / 20 / 24 / 32 / 40 / 48 / 64 / 80px (larger sizes for hero/section headings)

## Spacing Scale
Use a 4px base unit. Common values: 4, 8, 12, 16, 24, 32, 48, 64, 96, 128px
Generous spacing between sections — let the design breathe.

## Border Radius
- Small (inputs, chips): 4px
- Medium (cards, buttons): 8px
- Large (modals, containers): 16px
- Full (avatars, pills): 9999px

## Shadows
- Subtle: 0 1px 2px rgba(4, 7, 7, 0.05)
- Medium: 0 4px 12px rgba(4, 7, 7, 0.08)
- Strong: 0 8px 24px rgba(4, 7, 7, 0.12)

## Component Patterns
- Buttons: 16px vertical / 32px horizontal padding, 48px height, background #D56C3F with #FFF8E5 text, hover darkens to #CB673D. Primary CTA buttons are bold and obvious.
- Inputs: 48px height, 16px padding, 2px border, focus border uses accent #D56C3F
- Cards: 24px padding, subtle shadow, medium radius
- Sections: full-width, generous vertical padding (96-128px), scroll-triggered fade/slide animations

## Animation Guidelines
- Scroll-triggered reveals (fade up, slide in) for sections and key content
- Subtle parallax on background elements or textures
- Smooth transitions on hover states (200-300ms ease)
- No bouncy/playful animations — keep it confident and measured
- Consider a subtle animated background texture (noise, grain, or floating particles)

## Rules
1. Never introduce colors outside this palette
2. Always use the spacing scale — no arbitrary values
3. Maintain consistent border radius per element type
4. When in doubt, add more whitespace
5. No gradients unless specifically approved
6. No emoji in UI elements
7. Photography and product images should do the heavy lifting for visual interest
8. Keep it clean — the cream + black + burnt orange is the identity
9. Every section should feel intentional — no filler content
10. The site tells a story as you scroll
